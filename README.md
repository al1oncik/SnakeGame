# SnakeGame
Classic 2d snake game with Python and Pygame

# Install:
  
  sudo apt install python3 pip git
  
  pip install pygame
  
  git clone https://github.com/al1oncik/SnakeGame.git
  
  cd SnakeGame
  
  python3 main.py



# Controls:
  
  Movement -> w,a,s,d

  r -> replay

  q -> quit
